Projeto

Factorio Server Manager é um projeto open-source criado para facilitar a instalação, gerenciamento e administração de servidores dedicados do Factorio.

  • Dashboard
  • Gerenciamento de Saves
  • Configuração visual
  • Logs
  • Instalação automática do servidor
  • Docker Ready
